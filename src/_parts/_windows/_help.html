📘 Longchamp Budget

🔰 Présentation Générale

Longchamp Budget est un logiciel de création de budget prévisionnel conçu spécialement pour les groupes Scouts et Guides de France.
Développé par/pour le Groupe Longchamp à Marseille, il permet une gestion précise du budget annuel d’un groupe SGDF.

  • Portable : aucun besoin d’installation (exécutable autonome).
  • Compatibilité : Windows, macOS, Linux.
  • Sauvegarde automatique : chaque action modifie directement le fichier actif, aucune action manuelle n'est requise.

📂 Fichier de travail

Au lancement du logiciel, en haut à gauche, vous devez :

  • Créer un nouveau fichier .lb,
  • Ou ouvrir un fichier existant.

Les tableaux de bord Unités, Dépenses, QF et Budget (en haut à droite) deviennent accessibles uniquement lorsqu’un fichier est ouvert.

🧭 Navigation principale

Le logiciel se divise en quatre tableaux de bord :

  1. Unités
  2. Dépenses
  3. QF
  4. Budget

Note importante: Lors de votre saisie, si la bordure du champ actif devient rouge, cela signifie que votre saisie est invalide et ne sera pas prise en compte.
Il faudra saisir une valeur correcte.

🏕️ Écran « Unités »

Permet la gestion des unités du groupe : Farfadets, Louveteaux-Jeannettes, Scouts-Guides, etc.

Fonctionnalités :

  • Ajouter, modifier, supprimer des unités.
  • Définir le nombre d’enfants/ados et de chefs/cheftaines.
  • Choisir une couleur représentative pour chaque unité.
  • Réorganiser les unités par glisser-déposer ().

Règles de gestion :

  • Le nom d'une unité doit être unique.
  • Une unité ne peut pas être supprimée si elle a déjà des dépenses associées dans le budget.
  • Une unité spéciale appelée "Groupe" est présente par défaut et non supprimable :
    • Son nombre d’enfants et chefs-cheftaines est la somme de toutes les unités du fichier.

💸 Écran « Dépenses »

Permet de définir les postes de dépenses applicables aux différentes unités.

Création / Édition :

  • Nom (obligatoire)
  • Prix unitaire (obligatoire)
  • Taux de prise en charge par les unités (obligatoire, par défaut à 100%)
  • Description (optionnelle)
  • Association à au moins une unité (obligatoire)

Taux de prise en charge :

  • 100% → L’unité prend la totalité de la dépense.
  • < 100% → La part restante est répartie équitablement entre tous les enfants du groupe.

Gestion :

  • Déplacement des dépenses par glisser-déposer ().
  • Suppression et dissociation dépense/unité(s) interdites si la dépense est déjà utilisée dans une unité.

💰 Écran « QF »

Permet de définir les quotients familiaux, leurs coeff, les frais de commission en ligne ( coeff + montant ) et définir le nombre d'enfants/ados pris en compte dans le calcul de chaque quotient familial.

Création / Édition :

  • Nom (obligatoire)
  • Montant de la cotisation nationale (obligatoire)
  • Coefficient multiplicateur (obligatoire)
  • Coefficient pour frais de commission en ligne (obligatoire)
  • Montant fixe pour frais de commission en ligne (obligatoire)

Il est possible de réorganiser les QF par glisser-déposer ().

Répartir les quotients familiaux par unité :

Ce module sert à indiquer le nombre d'enfants pris en compte dans le calcul du quotient familial par unité.
Ces données sont cruciales, car elles servent de base pour déterminer le montant des cotisations.
Il est donc essentiel que le total saisi soit le plus proche possible du nombre d'enfants de l'unité, l'idéal étant qu'il soit égal.
Plus les informations seront exactes, plus les montants calculés seront précis et justes .

Formule de calcul : 

Tarif unitaire moyen pondéré (par unité) =
(Cotisation unité par enfant x Nombre d'enfants)
Total de (Nombre d'enfants par tranche QF x Coefficient multiplicateur de la tranche QF)

C'est sur la base de ce tarif unitaire moyen pondéré, que seront appliqués les coefficients multiplicateurs des tranches QF

📊 Écran « Budget »

L’espace principal pour bâtir et ajuster le budget de chaque unité.

Structure :

  • Les unités sont affichées sous forme d’accordéons.
  • Chaque unité comporte :
    • À gauche : liste des dépenses disponibles (association dépense/unité(s) opérée sur l'écran "Dépenses") .
    • À droite : tableau des dépenses intégrées au budget.

Note importante: Gardez à l'esprit qu'un(e) chef/cheftaine consomme mais ne paie pas.
La part des chefs/cheftaines est comptabilisée dans les dépenses... qui sont divisées uniquement par le nombre d'enfants/ados.

Ajouter une dépense :

Cliquez sur le bouton + en face d'une dépense → elle est ajoutée au tableau avec les valeurs par défaut.

Modifier une dépense :

  • Chaque valeur peut être surchargée individuellement.
  • Pour changer une valeur globalement, modifiez-la dans l’écran Dépenses après suppression des surcharges.

Formule de calcul : 

Total = Prix unitaire x Occurrences x (Enfants/Ados + Chefs/Cheftaines) x Taux de prise en charge

Autres fonctionnalités :

  • Glisser-déposer (), duplication, suppression, commentaires
  • Filtrage par nom de dépense (surlignage par couleur au clic sur le libellé de la dépense ( pas sur le bouton ))

🔍 Recherche

Appuyez sur le lien dans le bandeau à droite pour afficher/faire disparaître la zone de texte de recherche en bas de page.
Chaque saisie déclenche la recherche dans la page courante.

💡 Cas pratiques

📌 Ajouter une dépense classique

Ex : 6 goûters pour les Farfadets

  1. Sur l'écran "Dépenses", créez une dépense "Gouter" et associez la à l'unité Farfadets.
  2. Sur l'écran "Budget", ajoutez la dépense "Gouter" via le bouton +.
  3. Modifiez les occurrences à 6.

🏕️ Dépense unique

Ex : Achat d’une tente

  • Créer une dépense "Tente" et l’associer au Groupe.
  • Définir : Occurrences: 1, Enfants: 1, Chefs/Cheftaines: 0 (cf Formule de calcul).

💝 Gérer un don

  • Créer une dépense avec prix unitaire négatif (ex: -100).
  • Associer au Groupe, Occurrences: 1, Enfants: 1, Chefs/Cheftaines: 0 (cf Formule de calcul).

📤 Export Excel

Cliquez sur "Exporter vers Excel" pour générer un fichier .xlsx avec un onglet par unité, un onglet pour les QF et un onglet Balance

  • Le fichier est créé dans le même dossier que le fichier .lb.
  • Chaque export Excel écrase le fichier précédent.
  • Les calculs avancés restent dans le logiciel, l’export est simplifié.

🔄 Préparation du budget pour l'année suivante

  • Copier le fichier .lb et modifier le nom (ex: budget-2024.lb → budget-2025.lb).
  • Ouvrir le fichier budget-2025.lb.
  • Mettre à jour effectifs, prix unitaires, occurences...

📝 Bonnes pratiques

  • Créez toutes les dépenses typiques à l’avance.
  • Commentez les lignes particulières.
  • Évitez les surcharges de valeur sauf cas exceptionnel.

📎 Extension de fichier

Les fichiers Longchamp Budget utilisent l’extension : .lb
